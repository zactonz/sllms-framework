# Local Plugins

This folder is for deployment-specific local plugins.

Use it for:

- private tools that belong to one app or team
- fast local prototyping
- experiments before you split a reusable tool into its own repo

Reusable child repos should live outside the core `sllms-framework` repo.

See:

- `examples/plugins/`
- `templates/sllms_tool_pack_template/`
